# AI Health & Wellbeing Awareness System

## The main purpose of this project is to detect health on early symptoms,risks that helps to save lives through timely awareness and guidance.

## Description
An AI-based system that predicts diseases, detects health risks, and promotes wellbeing through awareness and guidance using machine learning.

# Objective
- Predict diseases based on symptoms  
- Detect emergency situations  
- Provide health awareness  
- Analyze student stress levels


# Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Decision Tree Algorithm  


# Features
- Disease prediction using symptoms  
- Emergency alert system 
- Cancer awareness based on symptoms  
- Student stress level detection  
- Guidance and awareness messages 

# Requirements
-Python 3.8 or higher

# How It Works
1. User enters symptoms  
2. Data is processed  
3. Model predicts disease  
4. System checks for serious conditions  
5. Alerts and suggestions are displayed  

## How to Run
pip install pandas and scikit-learn
python AIML_heath.py
